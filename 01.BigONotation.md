# Type of Big O Notation, the measures the worst-case complexity of the algorithm / Space & Time Complexity
https://github.com/mcholismalik/big-o-notation

1. O(1)
    - Constant Time
    - Calculate
2. O(n)
    - Linear Time
    - Loop
3. O(n^2)
    - Quadratic Time
    - Matrix
4. O(log n)
    - Logaritmic Time
    - Binary Search
5. O(n log n) 
    - Linearitmic Time
    - Merge Sort
